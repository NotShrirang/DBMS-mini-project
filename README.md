# Recipe Book

### Tech stack: 
- Front end: HTML, Bootstrap<br>
- Backend: Django<br>
- Database: sqlite3<br>
- Connector: <a href="https://github.com/NotShrirang/pydbops">pydbops</a><br>

### Clone this repo:

```
git clone https://github.com/NotShrirang/DBMS-mini-project.git
```

### Run this website:

```
python manage.py runserver
```
